## Projects

**🔐 [Zero Trust Architecture Lab](https://github.com/Aarib26/Advanced-Security-Infrastructure---ZTA)**
Identity-aware access with Keycloak, eBPF microsegmentation with Cilium, centralized security monitoring with the ELK stack. Aligned to NIST SP 800-207. Built, broken, and rebuilt until every layer was actually understood — not a tutorial followed once.

**🧪 [Selenium + Jenkins CI Pipeline](https://github.com/Aarib26/Selenium_test_rep-)**
Automated browser testing with Selenium and Pytest, wired into a Jenkins pipeline for CI/CD. Covers both the Freestyle and Jenkinsfile pipeline approaches.

**⚙️ [CLI Data Processor](https://github.com/Aarib26/Simple-CLI-Data-Processor)**
A Python command-line tool that pulls JSON from a public API and saves it to disk — built to practice real production patterns: `argparse`-based CLI input, structured logging instead of print statements, and proper error handling.
